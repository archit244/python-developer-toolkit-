# Python Developer Toolkit

A collection of beginner-friendly command-line utilities built to strengthen my Python fundamentals through hands-on projects.

Instead of following tutorials, I focused on building small tools that cover file handling, recursion, JSON, hashing, data structures, and program design.

## Projects

| Project | Description |
|---------|-------------|
| 🔐 Password Generator | Generate secure random passwords with customizable options. |
| 💰 Expense Tracker | Store, manage and analyze expenses using JSON. |
| 📂 File Organizer | Automatically organize files into folders based on file type. |
| 📦 Folder Backup Utility | Recursively create backups while preserving folder structure. |
| 📝 Bulk File Renamer | Rename multiple files using custom prefixes and numbering. |
| 🔍 Markdown Note Search | Search Markdown files and display matching lines with line numbers. |
| 🧾 JSON Formatter | Validate and pretty-print JSON files. |
| 🧬 Duplicate File Finder | Detect duplicate files using SHA-256 hashing and optionally remove duplicate copies while keeping the oldest file. |
| 📊 Directory Size Analyzer | Recursively analyze directory sizes and display a hierarchical breakdown. |

## Why this repository?

The goal wasn't to build production-ready applications.

It was to get comfortable with Python fundamentals by solving small, practical problems instead of only reading documentation or watching tutorials.

Each project focuses on a different concept while keeping the implementation simple and easy to understand.


Learning by building. 🚀
