# 🔐 Password Generator

A simple command-line Password Generator built in Python that creates secure random passwords based on user preferences.

## ✨ Features

- Generate passwords of any length
- Optional uppercase letters
- Optional numbers
- Optional special characters
- Input validation for all user inputs

## 🛠️ Tech Stack

- Python 3
- `random` module

## 🚀 Getting Started

Run the application:

```bash
python3 main.py
```

- String manipulation
- Input validation
- Random number generation
- Conditional statements
