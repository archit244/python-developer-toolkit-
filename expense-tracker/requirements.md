# Requirements

## Objective

Build a command-line application that allows users to manage their daily expenses.

---

## Functional Requirements

The application should allow the user to:

- Add a new expense
- View all expenses
- Search expenses
- Update an existing expense
- Delete an expense
- View expense statistics
- Exit the application

---

## Expense Fields

Each expense should contain:

- Name
- Amount
- Category
- Date

---

## Validation

- Amount must be greater than 0.
- User input should be validated.
- Required fields cannot be empty.

---

## Scope (Version 1)

This version is a command-line application only.

No authentication.

No database.

No GUI.

No cloud synchronization.
